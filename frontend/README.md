# Glassy UI Design - Frontend

This is the frontend application for the Product Comparison Agent, built with Next.js and React.

## Running the code

Run `npm install` to install the dependencies.

Run `npm run dev` to start the development server.

The application will be available at `http://localhost:3000`.

## Project Structure

- `app/` - Next.js App Router pages and layouts
- `src/` - React components and utilities
- `public/` - Static assets

## Features

- Product comparison interface
- Search functionality with loading states
- Product detail modals with full-page layout
- Responsive design with Tailwind CSS
- Smooth animations and transitions
